Phase 17 — Rigor

Is It Real or Random?

Machine learning models can find patterns even in random noise. Statistical validation is the suite of tests that separate genuine biological signals from spurious correlations. Phase 17 covers group-level biomarker analysis, fairness evaluation across demographic subgroups, and multivariate analysis to quantify the added value of imaging features over clinical data alone.

Null Hypothesis

"There is no difference between cohorts for this biomarker." Statistical tests tell us whether we can reject that statement with confidence.

Why Statistical Validation?

Consider a biomarker like "FAZ area." We compute the mean FAZ area for RETINORM (control) and MRCC (cardiac surgery / CAD) patients. The MRCC mean is higher. Is this a real difference or just random variation because we happened to sample a few MRCC patients with large FAZs?

Statistical tests answer this question by quantifying how likely the observed difference would be if there were truly no difference between the groups (the "null hypothesis"). A small p-value (typically < 0.05) means we can reject the null and conclude the difference is statistically significant.

But significance is not the whole story. A very large sample can make a tiny, clinically meaningless difference statistically significant. That is why we also report effect sizes (Cohen's d, rank-biserial correlation) that tell us how big the difference is.

For beginners: Think of a statistical test as a lie detector for your data. It tells you: "The difference you see between groups -- is it real, or is it just noise?" A p-value of 0.03 means: "There is a 3% chance that this difference would appear if there were actually no difference." We usually call p < 0.05 "statistically significant."

Group-Level Analysis (Biomarkers and Clinical Features)

We analyze each clinical feature and each OCTA biomarker (from both the commercial device and our Phase 8 generated masks) to determine which ones differ significantly between cohorts. This serves two purposes: it validates known physiological associations (e.g., vascular density differences between controls and CAD patients) and it generates hypotheses for the explainability analysis.

For pairwise comparisons (e.g., RETINORM vs MRCC, RETINORM vs FAMILIPO, etc.), we apply Bonferroni correction: if we run 6 pairwise tests (4 cohorts choose 2), the significance threshold becomes 0.05/6 = 0.0083. This prevents false positives from multiple testing.

Critical Distinction: Clinical vs. Generated Biomarkers

We keep clinical features and generated biomarkers in separate analyses throughout. The clinical features (age, sex, BMI, etc.) are measured directly. The generated biomarkers (vessel density, FAZ area, etc.) are extracted from Phase 8 pseudo-masks. Conflating them would confuse the reader about which findings come from measured data versus algorithmically derived quantities. Any statistical finding involving generated biomarkers must state the Phase 8 mask_source used.

Fairness Analysis

A model that works well for one demographic group but fails for another is not just scientifically incomplete -- it is potentially harmful in clinical deployment. Phase 17 includes a systematic fairness audit.

Subgroups We Evaluate

  • Sex: Male vs. female. Medical AI has historically underperformed for women due to underrepresentation in training data.
  • Age: Quartiles (<50, 50-60, 60-70, >70). OCTA biomarkers change with age, so the model might learn age confounds rather than disease patterns.
  • Diabetes: Present vs. absent. Diabetes causes retinal microvascular changes that could dominate other disease signals.
  • Bilateral vs. unilateral: Patients with one eye vs. both eyes. Unilateral patients might differ systematically (e.g., the eye was unavailable due to disease severity).

What We Measure

For each subgroup, we compute the same primary metrics (macro-AUC, macro-F1, balanced accuracy) and compare them to the overall performance. A significant degradation in any subgroup (p < 0.05 for the difference, with adequate sample size) is a fairness concern that must be reported and discussed.

Important: Fairness analysis can only flag issues, not fix them. If we find that the model performs worse for women, the next project iteration would need to address this with balanced sampling, data augmentation, or fairness-constrained training objectives.

Multivariate Analysis

The final component of Phase 17 asks: "Does adding imaging data improve prediction beyond what clinical features alone can do?" This is a critical question for the paper's contribution.

Logistic Regression (Clinical Baseline)

We fit a one-vs-rest logistic regression model using only clinical features (age, sex, BMI, smoking, hypertension, diabetes, dyslipidemia). The AUC of this model represents the "clinical ceiling" -- the best we can do without any imaging data.

Partial Correlation Analysis

For each clinical feature, we compute the partial correlation between that feature and the model's predicted disease probability, controlling for age and sex. This tells us which clinical features independently drive predictions, after removing the effects of age and sex (which are common confounds in retinal research).

For example: if the model assigns higher CAD probability to smokers even after controlling for age and sex, that is evidence the model has learned a clinically valid association. If the smoking signal disappears after controlling for age, the model might just be detecting "older patients smoke more."

The Key Result

The central finding from this phase: the full multimodal model significantly outperforms the clinical-only logistic regression baseline. The magnitude of this improvement (delta AUC with 95% CI) quantifies the added value of OCTA imaging for CAD risk stratification. If the delta is small, the paper's emphasis shifts away from "imaging is essential" toward "imaging provides marginal improvement in specific subgroups."

Glossary for Beginners

p-value
The probability of observing the data (or more extreme) if the null hypothesis were true. A p-value of 0.01 means there is a 1% chance of seeing this difference if there were actually no difference. Conventionally, p < 0.05 is called "statistically significant."
Mann-Whitney U Test
A non-parametric test that compares two groups without assuming the data follows a normal distribution. Works by ranking all values and comparing the sum of ranks between groups.
ANOVA
Analysis of Variance. Tests whether the means of three or more groups are equal. Assumes normally distributed data with equal variance across groups.
Bonferroni Correction
Adjusts the significance threshold when running multiple tests. If you run 10 tests and want to keep the overall false positive rate at 5%, each test must meet p < 0.005 (0.05 / 10).
Partial Correlation
The correlation between two variables after removing the effect of one or more confounding variables. For example, the correlation between smoking and CAD probability after removing the effect of age.
Effect Size
A standardized measure of how large a difference is, independent of sample size. Cohen's d = 0.2 is "small," 0.5 is "medium," 0.8 is "large." Unlike p-values, effect sizes are not affected by sample size.
